Rate post|page with a positive meter

== Description ==

Rate post|page with a positive meter -> information automatically goes to ambooba.com|ambuba.ru|geroszinios.lt site where top post will become articles there

== Installation ==

1. Upload `positive-article-plugin.php ` to the `/wp-content/plugins/` directory directly or use `Plugins` -> `Add New` -> `Upload` for uploading
2. Activate the plugin through the 'Plugins' menu
3. Change settings in `Positive Article` menu
